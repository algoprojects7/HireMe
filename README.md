# LabourLink: AI-Powered Unorganized Employee Management System

LabourLink is a comprehensive, multi-tenant SaaS platform designed to bridge the gap between unorganized workers and those who need their services.

## 🚀 Quick Start
1. **Clone & Install**: `npm install`
2. **Setup Database**: Configure `.env` and run `npm run db:generate -w @repo/database`
3. **Start Development**: `npm run dev`
